#Travel_backend
# How to start.






Clone the project

```bash
  git clone https://github.com/sabinbhattaraii/Travel_backend.git
```

Go to the project directory

```bash
  cd Travel_backend
```

Install dependencies

```bash
  npm install
```
# Run Locally

## Run express server
```bash
  npm run dev
```

## Run hosted backend
``` 
https://travel-backend-cyan.vercel.app/
```
